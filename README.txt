1.  Clone this repo:
    git clone https://github.com/DavidSkates/database_website.git

2. Request .env file from me.

3.  Make sure you have PHP installed:
    php -v

4.  If not installed, install it:
    brew install php

5.  Run this to start application:
    Mac:
    open "http://localhost:8000/index.html" && php -S localhost:8000

    Windows:
    start http://localhost:8000/index.html && php -S localhost:8000
